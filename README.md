# 📊 Mobile Sales Analysis Dashboard – Power BI

This project is a **Power BI dashboard** created to analyze mobile phone sales data and generate meaningful business insights through interactive visualizations.

The dashboard highlights sales performance, transaction trends, customer ratings, and geographic distribution of sales across India.

---

## 🚀 Project Overview

The goal of this project is to transform raw mobile sales data into a **visually interactive dashboard** that helps understand:

- Overall sales performance
- Monthly sales trends
- Total quantity sold
- Customer rating distribution
- Payment method usage
- Sales performance by mobile model
- Geographic distribution of sales

---

## 📌 Dashboard Features

### 1️⃣ Sales Overview
- **Total Sales:** 769M+
- **Total Quantity Sold:** 19K
- **Average Sales:** 40K
- **Total Transactions:** 4K

### 2️⃣ Monthly Sales Trends
Line chart showing how total quantity changes month-by-month.

### 3️⃣ Payment Method Analysis
Breakdown of transactions using:
- UPI
- Debit Card
- Credit Card
- Cash

### 4️⃣ Customer Ratings Analysis
Visualization of customer satisfaction levels (1–5 ratings).

### 5️⃣ Geographic Sales Distribution
Map visualization showing mobile sales across major cities in India.

### 6️⃣ Model-wise Performance
Table displaying:
- Mobile model
- Total quantity sold
- Total transactions
- Total sales
